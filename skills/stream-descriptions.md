This is the writing guide for AI-drafted stream descriptions. It rides along whenever a description is generated or edited — from the Plan Stream form for an upcoming broadcast, or from a past stream's details page for a finished one. Edit it to change the style every description follows.

## Voice

- Written in the streamer's voice to their audience — first person, direct, energetic but honest.
- No clickbait, no filler ("in this video I will…"), no meta-talk about plans, episodes, outlines, or prompts.

## Planned streams (upcoming broadcasts)

- The description announces what's coming: 2-4 short paragraphs, or a brief intro plus a bulleted run-of-show.
- Continue naturally from previous episodes: pick up loose threads, unfinished work, and open questions, and propose the concrete next items to cover.
- Forward-looking language belongs here ("we'll", "join in", "live on…").
- Plain markdown is fine — these render inside the app.

## Past streams (recordings published on YouTube)

- The description goes onto the YouTube video, so write it for YouTube search and recommendations:
  - Front-load the hook — the first one or two sentences carry the main keywords; they show above the fold and weigh heaviest in search.
  - Naturally include every keyword and phrase someone would search to find this content: tools, technologies, project and product names, activities, problems solved. Woven into sentences, never a keyword dump.
  - Summarize what actually happened on the stream (grounded in its outline), then add a "Chapters" block from the outline's timestamps — one per line in YouTube's format (`0:00 Intro` first, then `H:MM:SS Title`).
  - Close with 2-3 relevant #hashtags on their own line.
- Plain text only: no markdown syntax (`**bold**`, `##`, `[links](…)`) — YouTube renders descriptions literally. Bare URLs are fine; YouTube links them.
- Past framing: the stream already happened. No "join me live", no countdowns.

## Links

- When brand links are provided, include the relevant ones (bare URLs) near the end — URLs verbatim, only from that list, never invented.
